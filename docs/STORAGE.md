# File Storage & Metadata

| Type                 | Path                                      |
| -------------------- | ----------------------------------------- |
| Single image uploads | `assets/images/uploaded/single`           |
| Group image uploads  | `assets/images/uploaded/group`            |
| Generated images     | `assets/images/generated/YYYYMMDD_hhmmss` |
| Search uploads       | `assets/images/search_uploads`            |
| Audio Recorder       | `assets/audios/user1/`                    |

All images include EXIF `ImageUniqueId` as `TZP-XXX` (UUID-based).
